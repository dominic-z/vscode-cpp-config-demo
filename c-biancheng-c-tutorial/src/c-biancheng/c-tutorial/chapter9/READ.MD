# 指针变量和变量

指针变量本身也是一种变量，例如
```cpp
int a = 15;
int *p = &a;
```
其中a和p都是变量，都是内存中数据的助记符，只不过p是一个指向int的指针的助记符，记的是一个地址；而a是一个int的助记符，记的是内存中的int数据，所以可以理解为a和*p（这里的*不是定义时候的*，而是取值符）指的都是那个int
可以对p指向的数据进行赋值，就相当于把p指向的内存数据修改了
```cpp
*p=5
```
